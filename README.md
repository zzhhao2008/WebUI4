# WebUI4 ZSV WebUI4.0

## ZSV WebUI V4.0

### 基本参数
1. 前端：Bootstrap5.1.3
2. 后端 ：PHP 7.3.4NTS
3. LA\*P WA\*P支持
4. 重路由

### 更新日志：
#### 2023-11-22更新
1. 增加View-message 消息弹窗组件，同步JS与PHP函数
2. 在help中放置demo
3. 增加View-alert 提示框组件。一段时间后自动删除。
4. 将View:import默认防止在页面最后减少冗杂操作
5. 在标准CSS库中添加`.top-pos`和`.right-pos`两个组件
6. 增加前端JS渲染控制（目前仅消息框），辅助前后端分离设计
##### 已知问题
1. 消息框的JS和PHP函数参数顺序略有不同，其中PHP为原生，JS为转译
2. 没有动画
